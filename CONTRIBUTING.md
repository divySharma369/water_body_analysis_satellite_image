# Contributing Guidelines

We welcome contributions to the Water Body Segmentation project! Here's how you can help:

## 1. Reporting Bugs
If you find a bug, please create an issue detailing the problem, your environment, and steps to reproduce.

## 2. Suggesting Enhancements
We are always looking to improve our models and UI. If you have an idea, feel free to submit an issue outlining your proposal.

## 3. Submitting Pull Requests
- Fork the repository.
- Create a new branch for your feature or bug fix: `git checkout -b feature/my-new-feature`
- Commit your changes: `git commit -m 'Add some feature'`
- Push to the branch: `git push origin feature/my-new-feature`
- Open a Pull Request.

Please ensure your code follows standard Python conventions and add comments where necessary. If you add new functionality, please try to include a test in the `tests/` directory.

Thank you for contributing!
